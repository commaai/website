Each vehicle has a specific harness based on make, model and year. Each kit includes the following:

  • comma power v3
  • harness box
  • harness connector

Exceptions to the above include:

Ford Q4, Nissan A, Nissan B, Rivian A, Tesla A, and Tesla B

  • comma power v3
  • harness box
  • harness connector
  • long OBD-C cable (9.5 ft)
  • USB-C coupler

GM

  • harness box
  • harness connector

OBD-II

  • long OBD-C cable (9.5 ft)
  • OBD-II connector
  • USB-C coupler

VW J533

  • harness box
  • harness connector
  • long OBD-C cable (9.5 ft)
  • USB-C coupler
